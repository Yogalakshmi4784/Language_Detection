About the Dataset:
======================

It's a small language detection dataset. This dataset consists of text details for 17 different languages, ie, you will be able to create an NLP model for predicting 17 different language..

Languages:
===========

1) English
2) Malayalam
3) Hindi
4) Tamil
5) Kannada
6) French
8) Spanish
9) Portuguese
10) Italian
11) Russian
12) Sweedish
13) Dutch
14) Arabic
15) Turkish
16) German
17) Danish
18) Greek
